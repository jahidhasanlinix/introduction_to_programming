# Introduction to Programming in C

A chapter-wise introduction to programming using the C language. Each chapter focuses on one core concept with examples you can compile and run.

## Getting the code

```bash
git clone https://github.com/jahidhasanlinix/introduction_to_programming.git
cd introduction_to_programming
```

To compile and run a file (example):

```bash
gcc if_else.c -o if_else
./if_else
```

(On Windows with MinGW/GCC: use `if_else.exe` instead of `./if_else`.)

---

## Chapter 1: Conditionals (if-else)

**File:** [if_else.c](if_else.c)

Learn how to branch your program based on conditions using `if`, `else if`, and `else`. Covers simple comparisons, even/odd checks, and multi-way decisions.

---

## Chapter 2: Loops (for)

**File:** [for_loops.c](for_loops.c)

Use `for` loops to repeat a block of code a fixed number of times. Covers loop syntax (initialization, condition, increment) and common patterns like counting and repeating text.

---

## Chapter 3: Loops (do-while)

**File:** [do_while_loops.c](do_while_loops.c)

Execute code at least once, then repeat while a condition is true. Understand the difference between `while` and `do-while` and when to use each.

---

## Prerequisites

- A C compiler (e.g. GCC, Clang, or MinGW on Windows)
- A text editor or IDE

No prior programming experience is required; the course builds from the basics.
